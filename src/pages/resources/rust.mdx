# Rust

Rust est un langage de programmation compilé multi-paradigme opensource sous licence MIT et développé par Mozilla Research depuis 2010.
Il est conçu pour être "sûr, concurrent et pratique", en supportant les styles de programmation fonctionnelle et impérative, ainsi que la programmation orientée objet.
Rust est particulièrement adapté aux applications concurrentes, et aux programmes qui doivent résister aux pannes.

## Documentation

- [Rust Book](https://doc.rust-lang.org/book/) : documentation officielle du langage
- [Rust Reference](https://doc.rust-lang.org/reference/index.html) : documentation technique du langage
- [Rustling](https://github.com/rust-lang/rustlings) : exercices pour apprendre Rust

## Conventions

Rust recommande un certain nombre de conventions de codage pour améliorer la lisibilité du code et la compréhension du programme.
Ces conventions sont décrites dans la documentation officielle du langage :

- [Naming - Rust API Guidelines](https://rust-lang.github.io/api-guidelines/naming.html) : conventions de nommage
- [Comments - The Rust Reference](https://doc.rust-lang.org/reference/comments.html) : conventions de commentaires
- [Rust #5: Naming conventions](https://dev.to/cthutu/rust-5-naming-conventions-3cjf) : article résumant la documentation officielle
